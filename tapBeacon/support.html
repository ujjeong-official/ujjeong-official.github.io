FAQ

How do I restore Pro?
On the Paywall, tap Restore Purchases. Make sure you’re signed into the same Apple ID used to buy Pro.

Safe Walk is locked.
Safe Walk is part of Tap Beacon Pro. Unlock Pro to enable background check-in reminders and interval selection.

Messages aren’t sending automatically.
iOS requires user confirmation for SMS. Tap Beacon prepares the message and opens the Messages share sheet for you to send.

Location isn’t available.
Go to iOS Settings → Privacy & Security → Location Services, enable for Tap Beacon (preferably “Always”), then try again.

Refunds?
Purchases are processed by Apple. For refunds, use reportaproblem.apple.com.
